# 显示所有可用的开发工具

显示所有可用的开发工具

## 说明

按照以下格式在系统提示符下显示所有可用的工具：

1. **列出每个工具**及其函数签名

2. **添加每个工具的用途**作为后缀

3. **在工具之间使用双行分隔符**以提高可读性
4. **采用项目符号格式**以清晰组织

输出应帮助开发人员理解：
- 当前 Claude Code 会话中可用的工具
- 供参考的确切函数签名
- 每个工具的主要用途

示例格式：
```typescript
• functionName(parameters: Type): ReturnType - 工具的用途

• anotherFunction(params: ParamType): ResultType - 此工具的作用执行
```

此命令可用于：
- 快速参考可用功能
- 了解工具签名
- 规划用于特定任务的工具
